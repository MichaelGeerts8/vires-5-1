# Vires-5 Website

Een moderne webapplicatie gebouwd met React en Webpack.

## Projectstructuur

```
vires-5/
├── src/                 # Broncode
│   ├── components/      # React componenten
│   ├── pages/          # Pagina's
│   ├── styles/         # CSS/SCSS bestanden
│   └── utils/          # Hulpfuncties
├── public/             # Statische bestanden
│   ├── images/         # Afbeeldingen
│   └── fonts/          # Font bestanden
├── config/             # Configuratiebestanden
├── docs/               # Documentatie
├── tests/              # Tests
│   ├── unit/           # Unittests
│   └── integration/    # Integratietests
├── scripts/            # Utiliteitscripts
├── build/              # Gebouwde output
└── .github/            # GitHub workflows
```

## Installatie

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Testen

```bash
npm test
```
